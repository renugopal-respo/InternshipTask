#  Design Agency Website

A modern **Design Agency Landing Page** built using **React + Vite**.
This project demonstrates **component-based architecture, responsive design, basic animations, and SEO-friendly structure**.

The website showcases services, portfolio projects, and a contact form for user interaction.

---

##  Features

* Responsive Design (Mobile, Tablet, Desktop)
* React Component Architecture
* Navigation using **React Router**
* Modern UI with **CSS Modules**
* Basic UI Animations
* Contact Form with Success Message
* Portfolio Section with Project Cards
* SEO Metadata Implementation
* Mobile Hamburger Menu
* Clean and Minimal UI/UX

## ▶️ Running the Project

Clone the repository:

```
git clone https://github.com/your-username/design-agency.git
```

Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

The application will run at:

```
http://localhost:5173
```
